How To Run:
1: Ensure python is correctly installed
2: pip install pygame
3: Open PONG.py

Controls:
Up/Down Arrow Keys: Move
F11: Fullscreen
Escape: Small Screen
Space: Start game

Explanation Of Game:
This games difficult scales based on your score.
The more you are winning by, the harder it gets.
As such this game is tracked more by how high of a level you can get.
Instead of the usual keeping track of score.
